help - Print help about specific operators, settings or concepts

Syntax: h=HELP_SECTION

General:
    Prints a more detauled help section for specific concepts than the main
    help page. After printing the help text, the program is terminated with
    exit code 0, unless in REPL mode.

Examples:
    scr h=regex     # prints the help section of the `regex` operator
    scr h=r         # shorthand versions of operators can be used aswell
    scr h=types     # prints a summary of the data types used in scr
    scr h=main      # print the main help text
    scr h=h         # print this exact help section
