scr [FLAGS | SETTINGS | OPERATORS]...

Flags:
    -h | --help           Print this help and exit
    -v | --version        Print version info and exit

Settings:
    h=<cmd>               Print help about a Setting or Operator and exit

    bs=<N>                Batch size
    sbs=<N>               Stream buffer size
    j=<N>                 Max number of threads (default 1, use 0 for core count)
    lb=<yes|no|...>       Line Buffering Mode (see mode descriptions in `h=lb`)

    denc=<encoding>       Default text encoding (see encodings list in `h=enc`)
    ppenc=<yes|no>        Prefer parent document encodins over the set default
    fenc=<yes|no>         Force use of denc, despite any BOMs or <meta charset>

    sel=<driver>          Enable Selenium (see list of drivers in `h=sel`)
    sds=<strategy>        Selenium Download Stragegy (list in `h=sds`)

Operators (`h=<operator>` for details):
    p                     Print previous results to stdout
    f=<fstr>              Format text using rust format strings
    r[abdilmu]=<regex>    Apply a regular expression
    key=<IDENT>           Give a name to the previous column.

    [+]file[t]=<path>     Read in bytes (or text) from a file
    [+]stdin[t]           Read in bytes (or text) form stdin

    [+]seq=<B>,<E>,<S>    Produce a sequence of Integers
    enum=<B>,<E>,<S>      Insert one sequence element for every match

    [+]str<N>=<str>       Insert string data  (<N> times)
    [+]bytes<N>=<bytes>   Insert raw bytes    (<N> times)
    [+]int<N>=<int>       Insert an integer   (<N> times)

    split=<chainspec>     Forward matches to other chains
